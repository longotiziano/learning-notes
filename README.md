# Notes

Este directorio reúne mis apuntes, resúmenes y documentación personal relacionada con mi proceso de aprendizaje en áreas como:

- Análisis de Datos  
- Ingeniería de Datos  
- Python aplicado a datos  
- SQL  
- Big Data / Spark  
- Cloud (conceptos generales)  
- Otros temas técnicos de interés  

La finalidad de este espacio es centralizar toda mi documentación de estudio en un único lugar accesible desde cualquier dispositivo, aprovechando GitHub como entorno seguro y versionado.

## Contenido
Cada archivo dentro de `notes/` corresponde a un tema, módulo o curso que estoy estudiando.  
Los apuntes están redactados en mis propias palabras, incluyen conceptos clave, ejemplos prácticos, aclaraciones y buenas prácticas útiles para repasar más adelante.

## Actualización continua
Este directorio se actualiza regularmente a medida que avanzo con nuevos cursos o reviso material ya aprendido.  
La idea es mantener un registro vivo de mi formación en datos.

## Objetivo
- Consolidar conocimiento  
- Tener una referencia rápida mientras programo  
- Documentar mi progreso real  
- Usar GitHub como “cloud” de estudio y código
